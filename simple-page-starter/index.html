Web Development Skills Learn HTML HTML stands for Hypertext Markup Language. HTML is a collection of tags web designers use to build web pages. You also use HTML to add page content, including headings, paragraphs, hyperlinks, images, lists, etc. Check out the HTML tutorial on html.com. --------------------------------------------------------------------------- Learn CSS CSS stands for Cascading Style Sheets. Web designers use CSS to style web pages. You can use style sheets to specify the background color of a web page, font styles, font colors, borders and dimensions. CSS Grid and CSS Flexbox are two brand new techniques web designers use to create page layouts. Check out this CSS tutorial on MDN Web Docs. --------------------------------------------------------------------------- Learn JavaScript JavaScript is a language web designers use to make their web pages interactive and dynamic. Examples of common JavaScript tasks include: - add new HTML to a web page - change the existing content and styling of a web page - respond to user events (mouse clicks, key presses, resizing the browser window, etc.) Check out the JavaScript tutorial on javascript.info.
